# Simplified Code Block Syntax

You can now use a simpler syntax for executable code blocks:

## Basic Example

```code :language python :id "example1"
print("Hello, World!")
print(2 + 2)
```

## With Sessions

```code :language python :id "session-a" :session "demo"
x = 10
y = 20
```

```code :language python :id "session-b" :session "demo"
print(f"x + y = {x + y}")
```

```code :language python :id "session-c" :session "demo"
print(f"x * y = {x * y}")
```

## Lean4 Example

```code :language lean4 :id "lean-example"
def triple (x : Nat) : Nat := x + x + x

#eval triple 7
```

## Syntax

The format is:

\`\`\`code :language LANG :id "ID" :session "SESSION"
your code here
\`\`\`

- `:language` - The programming language (python, lean4, etc.)
- `:id` - Unique identifier for auto-execution
- `:session` - Optional session name to share state between blocks

Then run `npm run execute-code` to generate outputs!
